Welcome

The Climate Stress Tool is a web application for generating synthetic weather data using a stochastic weather generator.

Quick Start

  1. Load historical weather data either by selection a location on a map (Northeast US only) or uploading a file
  2. View timeseries plots of the historical weather data to confirm data were loaded correctly
  3. Configure the weather generator by setting the timespan and various adjustment factors
  4. Submit the simulation job to the server and wait for completion
  5. View and download weather generator results

About

The Climate Stress Tool was developed by Jeffrey D Walker, PhD and the Hydrosystem Research Group at UMass Amherst. This application provides a web-based interface to the weathergen R package, which is used to perform the simulation on a remote server.

The weather generator algorithm is based on Steinschneider and Brown (2013):
