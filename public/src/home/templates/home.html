Welcome to the Climate Stress Tool

The Climate Stress Tool is a web application for generating synthetic weather data using a stochastic weather generator.

Quick Start

  • Load some data
  • Configure the generator
  • Run a generator job
  • View and download results

See the Documentation page for more details.

About

The Climate Stress Tool was developed by Jeffrey D Walker, PhD and the Hydrosystem Research Group at UMass Amherst.
